# netflixCommunity (Toy Project)
* 스프링부트 & JPA를 기반으로 만든 커뮤니티 웹사이트
* 회원가입, 로그인/로그아웃, 게시물 조회 및 등록, 수정, 댓글, 좋아요 등의 기능 구현

## 프로젝트 개발 환경
* IDE: 인텔리제이
* JDK 11
* Spring Boot
* MySQL
* JPA
* gradle
